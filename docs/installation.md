# Guía de Instalación

## Requisitos Previos
- Node.js (v18 o superior)
- npm/yarn
- Base de datos (PostgreSQL recomendado)

## Pasos de Instalación

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Configurar variables de entorno
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env
# Configurar variables de entorno
npm run dev
```
